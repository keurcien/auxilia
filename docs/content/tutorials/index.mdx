# Tutorials

Practical, business-friendly walkthroughs showing how to put **auxilia** agents to work in real-world situations — beyond the chat UI.

These are designed to be accessible: you don't need to be a developer to follow along, but your IT or AI engineering team may need to help with the initial setup.

## Available tutorials

- **[Trigger agents over HTTP](/docs/tutorials/http-invoke)** — call an **auxilia** agent from any tool that can make a web request, without writing custom code
- **[Trigger agents from Slack](/docs/tutorials/slack)** — give your team a familiar way to talk to agents directly in Slack threads
- **[Embed an agent in a custom UI](/docs/tutorials/embed-custom-ui)** — put an **auxilia** agent inside your customer-support console, back-office dashboard, or Google Sheets add-on
- **[Use auxilia from n8n & Cloud Scheduler](/docs/tutorials/n8n-cloud-scheduler)** — run agents on a schedule, or plug them into automation workflows

## Something missing?

If you've built something with **auxilia** and want to share the recipe, head over to [Case Studies](/docs/case-studies) — we welcome contributions.
